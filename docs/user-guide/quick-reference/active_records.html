ActiveRecord Quick Reference

  • ActiveRecords are objects that represent a table or record in a table by abstracting the SQL and providing an object oriented interface
  • ActiveRecords will automatically map relationships for each table providing the database complies with the convention
[edit]

Features

  • ActiveRecords support composite primary keys
  • ActiveRecords support auto-increment primary keys
  • ActiveRecords support string based primary keys
  • ActiveRecords support lowercase and uppercase table/field names
  • ActiveRecords support table field prefixes
  • ActiveRecords support one to many relationships
  • ActiveRecords support many to one relationships
  • ActiveRecords support many to many relationships
  • ActiveRecords support one to self relationships (fishhook)
  • ActiveRecords support multiple relationships on the same table (recursive)
  • You may manually define the mapping for any table
[edit]

Limitations

  • Associative tables must use the same field names as the foreign table even if mapping is manually defined
  • The following methods do not work when multiple relationships to the same object exist:
    • BaseActiveRecord::removeObjectsByType()
[edit]

Database Convention

The ActiveRecords will attempt to map themselves automatically based on a database convention

  • all foreign keys must match the associated primary key
  • all field names may have an optional prefix of the table followed by an underscore

e.g. 

`users` ( `users_id`, `categories_id`, `users_name`, `users_age` )

`categories` ( `categories_id`, `categories_name` )

`groups` ( `groups_id`, `groups_name` )

`users_groups`( `users_id`, `groups_id` )
  • each category has many users (one to many)
  • each user may be associated with one or more groups (many to many)
[edit]

Static Methods

Apply to all records of this type All methods are protected and must be overridden as public to be used outside the object

[edit]

BaseActiveRecord::add()

create a new ActiveRecord of this type optional associative array of initial values

static protected BaseActiveRecord add ([ $args = array()])

example: 

$john = Users::add( array( 'name' => 'john' ));
[edit]

BaseActiveRecord::find()

find an existing ActiveRecord of this type requires associative array of values to search for returns null if no record found

static protected BaseActiveRecord find ([ $args = array()])

example: 

$jane = Users::find( array( 'name' => 'jane' ));
[edit]

BaseActiveRecord::findById()

find an existing ActiveRecord of this type requires id of record to search for returns null if no record found

static protected BaseActiveRecord find ([ $args = array()])

example: 

$myUser = Users::findById( 7 );
[edit]

BaseActiveRecord::first()

find the first ActiveRecord of this type returns null if no record found

static protected BaseActiveRecord first ()

example: 

$firstUser = Users::first();
[edit]

BaseActiveRecord::last()

find the last ActiveRecord of this type returns null if no record found

static protected BaseActiveRecord last ()

example: 

$lastUser = Users::last();


[edit]

BaseActiveRecord::findAll()

return array of ActiveRecords of this type

static protected ActiveRecordCollection findAll ()

example: 

foreach( Users::findAll() as $user ) {
  // each $user is an ActiveRecord object
}
[edit]

BaseActiveRecord::dataSet()

return Recordset containing all records of this type

static protected DataSet DataSet()

example: 

foreach( Users::dataSet()->rows as $user ) {
  // each $user is a DataSet row (array)
}
[edit]

Public Methods

Apply to a single instance of an ActiveRecord

[edit]

BaseActiveRecord::save()

save ActiveRecord (will write data to database adapter)

public bool save ()

example: 


$john = Users::add();
$john->save();
[edit]

BaseActiveRecord::delete()

delete ActiveRecord (will write data to database adapter)

public bool delete ()

example: 

$john = Users::find( array( 'name' => 'john' ));
$john->delete();
[edit]

BaseActiveRecord::deleteAll()

cascading delete delete ActiveRecord and all associated records (will write data to database adapter)

public bool deleteAll ()

example: 

$john = Users::find( array( 'name' => 'john' ));
$john->deleteAll();
[edit]

Static Properties

Apply to all records of this type

[edit]

BaseActiveRecord::db

instance of the ADOConnection object null on error

static ADOConnection $db

example: 

Users::add();
echo Users::$db->lastQuery;
[edit]

Public Properties

Apply to a single instance of an ActiveRecord

[edit]

BaseActiveRecord::row

array of ActiveRecord properties

array $row

example: 

$john = Users::add();
$john->row['name'] = 'John';
[edit]

BaseActiveRecord::dataAdapter

instance of the SqlDataAdapter object throws exception on error

protected SqlDataAdapter $dataAdapter

example: 

$john = Users::add();
echo $john->dataAdapter->lastQuery;
[edit]

BaseActiveRecord::table

readonly name of the abstracted database table

readonly string $table

example: 

$john = Users::add();
echo $john->table;
[edit]

BaseActiveRecord::pkey

readonly name of the primary key for the abstracted table

readonly string $pkey

example: 

$john = Users::add();
echo $john->pkey;
[edit]

BaseActiveRecord::prefix

readonly name of the table prefix for the abstracted table

readonly string $prefix

example: 

$john = Users::add();
echo $john->prefix;
[edit]

BaseActiveRecord::mappings

readonly array of the ActiveRecord mappings

readonly array $mappings

example: 

$john = Users::add();
print_r( $john->mappings );
[edit]

Virtural Methods

Apply to a single instance of an ActiveRecord

[edit]

BaseActiveRecord::getAllRecordsByType()

return array of all associated ActiveRecords by type

example: 

$john = Users::add();
foreach( $john->getAllGroupRecords() as $group ) {
  // $group is an ActiveRecord object associated to the User object
}
[edit]

BaseActiveRecord::getRecordsetByType()

return Recordset with all associated objects by type

example: 

$john = Users::add();
foreach( $john->getGroupRecordset()->rows as $group ) {
  // each $group is a Recordset row (array)
}
[edit]

BaseActiveRecord::getParentRecordByType()

return the associated parent ActiveRecord by type null if none found

example: 

$john = Users::add();
$category = $john->getParentCategoryRecord();
[edit]

BaseActiveRecord::addRecord()

associate an existing ActiveRecord requires instance of an ActiveRecord

example: 

$john = Users::add();
$john->addCategoryRecord( Categories::first() );
[edit]

BaseActiveRecord::removeAllRecordsByType()

remove all associated ActiveRecords by type

example: 

$john = Users::first();
$john->removeAllCategoryRecords();
[edit]

BaseActiveRecord::deleteAllRecordsByType()

delete all associated ActiveRecords by type (will write to database adapter)

example: 

$john = Users::first();
$john->deleteAllCategoryRecords();
[edit]

Virtural Properties

Apply to a single instance of an ActiveRecord

[edit]

BaseActiveRecord::$property

access to ActiveRecord property an error will be thrown if the property is not defined

public mixed {$property}

example: 

$john = Users::add();
$john->name = 'John';
[edit]

BaseActiveRecord[$property]

array access to ActiveRecord property an error will be thrown if the property is not defined

public [] {$property}

example: 

$john = Users::add();
$john['name'] = 'John';
