Controllers

The controller handles incoming requests from the HTTP server and is responsible for manipulating the model and dispatching the appropriate the view.

In this example the controller is assigning a variable to the view /app/views/blog.tpl during page load that contains all blog entries from the model Entries

<?php

    namespace MyBlog\Controllers;

    class Blog extends \MyBlog\ApplicationController
    {
        public function onPageLoad($sender, $args)
        {
            $this->page->assign('entries', \MyBlog\Models\Entries::all());
        }
    }
?>

Controller instances are located in /app/controllers
