Overview

PHPRum is an event driven object oriented HTML5/PHP5 web application framework that allows you to build web applications with as few lines of code as possible using concepts like Controllers, Views, Active records, Mappable web controls, Data binding, and Convention over Configuration. Rum was designed conceptually from the ground up to mimic the architecture of a desktop application using events, and Web controls. Rum is a full stack framework has been in development since 2003 and is released under the GNU license.

Rum applications can be built on any platform and does not require any developer toolkit, however IDE's like Netbeans/Eclipse are highly recommended for code completion.

Rum also includes a set of Windows/Mac/Linux command line deployment and database versioning tools, and source code template generators.

Features

  • Requires no external packages or libraries
  • No configuration
  • Native PHP templating
  • Auto class loading
  • Clean URL's
  • UI Web control library
  • Automated deployment over SSH
  • Drag and drop deployment
  • Database migration and versioning
  • Code generation
  • Integrated unit testing
  • Single point authentication
  • Namespace support
  • Database adapters
  • PDO support
