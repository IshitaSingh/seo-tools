Make

Make is a bundled command line tool that quickly generates source code based on pre defined templates. Make requires the PHP command line tool.

Usage

This example will generate a new blank controller class and view template in /app/controllers/listblogentries.php and /app/views/listblogentries.tpl

php make controller ListBlogEntries

This example will generate a new blank model class (and auto map the relationships) in /app/models/entries.class.php

php make model Entries
