Views

The view represents the user interface which consists of the HTML and all of the things the user can see and respond to on the screen, such as buttons, text fields, tables, etc... The view can send messages directly from the user to the model via the controller.

In this example the view is telling the browser to display a list of entries that have been sent from the controller.

<h1>My Blog</h1>

<?php foreach($entries as $entry) : ?>

<h3><?php echo htmlentities($entry["title"]) ?></h3>
<p><?php echo htmlentities($entry["body"]) ?></p>

<?php endforeach; ?>

<?php echo \Rum::link('Add a new entry', 'new_entry') ?>

Views are located in /app/views
