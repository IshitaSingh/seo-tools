Folder Structure

/app - contains all application specific files
/app/code - contains any user defined classes
/app/config/deploy - contains all deployment configuration files for each environment
/app/config/environments - contains environment specific configuration files
/app/config/migrations - contains database migration up/down scripts
/app/controllers - contains all controller classes
/app/layouts - contains all HTML layout files
/app/models - contains all active record and model classes
/app/plugins - contains any Rum plugin packages
/app/tests - contains all unit test scripts
/app/views - contains all view files
/logs - contains all log files
/public - contains any global HTML resources
/public/themes - contains all theme packages
/system - contains core Rum package
